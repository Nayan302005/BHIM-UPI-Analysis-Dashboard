# 📌 Project Overview

This project presents an interactive **BHIM UPI Transaction Analysis Dashboard** developed in **Power BI** to analyze digital payment performance across users, merchants, transaction types, locations, and payment outcomes.

The dashboard transforms raw transaction data into meaningful insights through dynamic visualizations and KPI tracking, helping stakeholders monitor transaction activity, identify trends, evaluate user behavior, and improve payment ecosystem performance.

# 🎯 Business Objectives

The dashboard was designed to answer key business questions:

- How many total transactions were processed?
- What is the total transaction value?
- What is the transaction success rate?
- Which states generate the highest transaction volume?
- Which cities contribute the most transaction value?
- Which transaction types are most frequently used?
- How do users and merchants perform across regions?
- What factors contribute to transaction failures?
- How can digital payment adoption be improved?

# 📊 Dashboard Pages

## 📊 Page 1: Executive Overview

This page provides a high-level summary of overall BHIM UPI performance.

### KPI Cards

- 🔄 Total Transactions
- 💰 Total Transaction Value
- ✅ Success Rate
- ❌ Failure Rate
- 👥 Active Users
- 🏪 Active Merchants

### Visualizations

#### 📈 Transaction Trend Analysis
Tracks transaction volume and value over time.

#### 🥧 Transaction Status Distribution
Shows the proportion of successful and failed transactions.

#### 📊 Transaction Type Analysis
Compares P2P and P2M transaction activity.

#### 🌍 State-wise Transaction Performance
Highlights top-performing states based on transaction volume and value.

### Slicers

- Date
- State
- Transaction Type
- Status

# 📈 Key Metrics Tracked

|----------|-------------|

| Total Transaction Value | Total payment value processed |
| Success Rate | Percentage of successful transactions |
